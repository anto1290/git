Branch adalah cabang dari repo yang kita buat

Branch dengan mundah membuat pararel dev

git branch melihat semua branch local

git branch --all melihat semua branch public

git branch name_branch membuat branch baru

git checkout name_branch berpindah dari branch 1 ke yang lain

git branch -d name_branch menghapus branch dengan memperhatikan kita tidak bisa mendelete dimana posisi branch yang digunakan dan yang akan didelete berada pada posisi yang sama

git merge name_branch mengabungkan 2 branch commit

conflict terjadi ketika kita merger 2 branch dimana posisi 2 branch melakukan perubahan posisi yang sama atau file yang sama. dan harus menyelesaikan conflict terlebih dahulu agar bisa merge
