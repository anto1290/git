Git init adalah syntax awal git untuk menginisialkan sembuah code repo

Git config untuk membuat config dalam seuatu repo dan konfigurasi bisa dari local maupun global biasanya diawalin user.email && user.name

Git status adalah untuk melihat status yang ada dalam repo. terdapat 3 untracked staging dan modified

git add untuk memasukan file ke area staging

git commit untuk memberi pesan apa yang sudah di ubah atau ditambah

git log untuk melihat riwayat commit

git clone untuk membuat copy repo server ke local

git remote untuk memantau client dan server repo agar selalu berkomunikasi

git pull untuk mengambil data dari server dan update ke local

git push untuk mengirim file kita/ branch ke server repo
